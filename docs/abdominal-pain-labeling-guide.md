# MedAsk 腹痛定向审核标签指南 v1

> 本指南用于18—65岁成人患者侧文本的语言标签审核，不用于疾病诊断、疾病概率、用药或治疗建议。遇到信息不足时选 `uncertain`，不得猜测。

## 1. 审核对象与顺序

只阅读队列中的 `title` 和 `ask`，不读取医生回答。建议按以下顺序独立判断：

1. 文本是否有效、主体和时间是否清楚。
2. 是否表达本次当前或近期腹部疼痛。
3. 最终主诉可否包含 `abdominal_pain`，以及是否还有其他主诉。
4. 咨询意图。
5. 是否出现需要未来风险规则拦截的明确表达。

## 2. 人工字段

### `human_is_valid`

- `yes`：患者侧文本足以判断语言标签。
- `no`：空模板、无意义短句、严重拼接或title/ask明显错配。
- `uncertain`：看似有信息，但主体、时间或上下文无法可靠判断。

### `human_current_symptom`

- `yes`：本次当前存在，或近期本次发作仍在描述中。
- `no`：明确否定、纯历史、纯假设，且本次没有当前症状。
- `uncertain`：如“前段时间腹痛”但没有说明现在是否仍有。

近期已缓解腹痛不标成当前仍存在；其本次事件可保留在最终主诉中，并在备注标记 `resolved`。

### `human_final_complaint`

允许多标签，使用竖线分隔。仅当文本明确表达腹部区域疼痛时加入 `abdominal_pain`。不要把疾病名称当主诉，也不要因为出现任意“痛”字就加入腹痛。

### `human_intent`

- `symptom_intake`：主要整理当前或近期本次症状。
- `diagnosed_followup`：明确已确诊、治疗、住院、出院或复查后的咨询。
- `medication_query`：主要询问吃什么、能否服用、剂量或药物作用。
- `report_interpretation`：主要询问检查、影像、化验指标含义。
- `hospital_or_cost`：主要询问医院、科室、医生或费用。
- `disease_knowledge`：主要询问定义、原因、表现、遗传或预防。
- `pediatric_or_pregnancy`：患者为儿童、孕产妇或哺乳期人群。
- `invalid_or_template`：文本无效、模板或明显错配。
- `uncertain`：无法可靠归类。

### `human_risk_expression`

- `yes`：文本明确、当前肯定地出现风险草案中的表达。
- `no`：没有出现，或明确否定/纯历史/已缓解/假设。
- `uncertain`：风险程度、主体或当前性不清楚。

该字段只标记语言表达，不推断疾病，也不代表完整临床分诊。

### `human_notes`

可为空。建议只记录必要的边界原因，例如 `resolved`、`title/ask冲突`、`仅腹胀无疼痛`、`主体不明`，不要写诊断或建议。

## 3. 腹痛正例边界

只有当前或近期本次腹部区域疼痛才进入 `abdominal_pain`，包括明确的腹痛、肚子疼、胃部疼、小腹疼、上腹疼或下腹疼。

合成正例：

- “今天开始肚子疼。” → 当前腹痛。
- “这两天上腹一直疼。” → 当前腹痛。
- “昨天小腹疼，现在仍有不适。” → 当前腹痛。

## 4. 不应自动标为腹痛

单纯腹胀、胀气、恶心、反酸、烧心、胃口差、食欲下降或泛化的“胃不舒服”，没有明确疼痛证据时不能加入 `abdominal_pain`。

腰痛、胸痛、头痛、痛经或经期不适也不能因为出现“痛”字映射为腹痛。若文本同时独立表达腹痛和其他主诉，保留多标签，不互相覆盖。

## 5. 否定、历史、已缓解与假设

- 明确否定：“没有腹痛”“肚子不疼” → `current_symptom=no`，不作为当前阳性腹痛。
- 纯历史：“去年有过腹痛，现在没事” → `current_symptom=no`。
- 近期已缓解：“昨天腹痛，现在已经好了” → 本次可进入信息整理，`current_symptom=no`，备注 `resolved`。
- 当前复发：“之前缓解，今天又开始腹痛” → 以最后的当前肯定为准，`current_symptom=yes`。
- 假设或知识：“如果以后腹痛怎么办”“腹痛有哪些原因” → 不进入当前症状预问诊。

## 6. 不支持人群与亲属主体

儿童、婴幼儿、明确小于18岁、孕妇、孕期、产后、恶露或哺乳期患者标记 `pediatric_or_pregnancy`，不并入当前成人流程。

亲属咨询本身不等于无效：成人亲属仍可按成人流程判断；儿童或孕产妇亲属仍进入不支持人群。主体不清楚时选择 `uncertain`。

## 7. 非症状预问诊意图

问药、报告解读、医院费用、疾病知识和已确诊随访即使提到腹痛，也应按主要咨询目的标记对应意图，不能仅凭关键词当成 `symptom_intake`。

## 8. 风险表达

以下只作为独立风险语言标签：当前突然剧烈腹痛、呕血、明显便血或黑便、明显腹胀且无法排便排气、腹痛同时晕厥或意识异常、腹痛同时严重呼吸困难。

必须同时判断当前性和否定语境。“没有呕血”“去年有过黑便”“担心以后突然腹痛”不能标为当前肯定风险。模糊表达优先 `uncertain`。

## 9. 一致性原则

- 不覆盖原始title或ask。
- 不把多主诉压成一个标签。
- 不把AI、候选规则或sampling_reason当作正确答案。
- 不生成疾病诊断、用药、剂量或治疗建议。
- 重大歧义无法通过本指南解决时，记录到notes并阻断开发放行。
