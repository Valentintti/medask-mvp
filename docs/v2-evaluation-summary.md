# MedAsk V2 头痛与头晕合成工程评测摘要

> 本文只记录人工合成案例上的工程指标，不代表临床准确率、诊断能力、治疗效果或医疗安全性。评测未使用真实患者原文、医生 answer 或真实 Provider。

## 数据集结构

- 总案例：132 条，四类主诉各 33 条。
- 本阶段可执行集：66 条（`headache` 33 条、`dizziness` 33 条）。
- 仅保留设计集：66 条（`abdominal_pain` 33 条、`chest_discomfort` 33 条），不计入生产通过率。
- 案例覆盖当前肯定、否定、历史、已缓解、假设、模糊、多主诉、风险、不支持人群、无效模板和槽位冲突。

## 通过与停止条件

上述核心安全和产品门槛均已达到，本阶段停止继续针对合成集调参。已知限制包括：规则词表不能覆盖全部自然语言；本结果不能外推到真实用户、真实 Provider 或任何临床效果；腹痛和胸部不适没有进入生产评测。
